Petunjuk Konfigurasi :

silahkan konfigurasi web server untuk pengaturan aplikasi e-apotek online.

anda harus mengubah lokasi directory "DocumentRoot" ke folder public untuk menjalankan aplikasi E-Apotek.

Contoh :

< VirtualHost *:80>

	ServerName example.com
	ServerAlias www.example.com
	ServerAdmin demo@example.com

	DocumentRoot /var/www/html/e-apotek/public/

	< Directory /var/www/html/e-apotek/public/>
		Options Indexes FollowSymLinks MultiViews
		AllowOverride All
		Order allow,deny
		allow from all
	< /Directory>

	ErrorLog ${APACHE_LOG_DIR}/error.log
	CustomLog ${APACHE_LOG_DIR}/access.log combined

< /VirtualHost>
